# Images Directory

This directory contains placeholder references for image assets. As per project requirements, we use online CDN images instead of local files.

## Image Sources Used:
- **Logo**: Online SVG/PNG from design resources
- **Hero Background**: Unsplash/Pexels CDN images
- **Station Placeholder**: Online placeholder images
- **Icons**: Lucide React, Heroicons, or FontAwesome icons

## Usage:
Images are referenced via online URLs in the components directly, not as local files.
